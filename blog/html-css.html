The difference between inline and inline-block

The 'display: inline' and 'display: inline-block' declarations make content display inline - that is, on the same line. If we want some content to display on the same line as other content, but would also like to be able to set a width, height, padding or margin, we can use inline-block.

By using inline-block, we gain some of the manipulation advantages of block content, while still displaying our content on the same line.
